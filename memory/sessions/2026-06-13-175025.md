## Decisions Made

- Stay Layer-4 (protocol-agnostic; SSH/HTTP/HTTPS pass through). nginx is inspiration only.
- Tokio async; minimal deps (tokio + serde/serde_json + anyhow); no clap, no chrono.
- Single binary, two subcommands; hand-rolled arg parsing.
- Optional `discovery` block — fall back to a static `target_ip` for "no complicated config".
- Hand-rolled UTC log timestamps (zero-dep), a deliberate change from the JS local-time logs.
- Relay is a manual `select!` loop (idle-reset timeout, half-close), not `copy_bidirectional`.
- Fixed three JS-era bugs: duplicate signal handlers, locale-fragile ETIMEDOUT match,
  IPv6/`::ffff:` allow-list normalization.
